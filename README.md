## Features

* ✨ Lightweight and optimized
* 🚀 Fast teleport requests
* 👥 Request players to teleport to you or teleport to them
* ❌ Cancel pending teleport requests
* ✅ Accept incoming requests
* 🚫 Deny incoming requests
* 💬 Clean and configurable messages
* ⚡ Designed for minimal performance impact

---

## Commands

| Command | Description |
|---------|-------------|
| `/tpa <player>` | Send a teleport request to another player. |
| `/tpahere <player>` | Request another player to teleport to you. |
| `/tpaccept` | Accept the most recent teleport request. |
| `/tpadeny` | Deny the most recent teleport request. |
| `/tpacancel` | Cancel your outgoing teleport request. |

---

## How It Works

1. Send a teleport request using `/tpa` or `/tpahere`.
2. The target player receives the request.
3. They can use `/tpaccept` to accept it.
4. Or use `/tpadeny` to deny it.
5. The sender can cancel a pending request at any time using `/tpacancel`.

---

## Why SimpleTps?

SimpleTps focuses on doing one thing well: player teleport requests.

Unlike larger essentials plugins, SimpleTps provides only the teleport request system, making it perfect for servers that want a lightweight, dedicated solution without unnecessary features.

---

## Configuration

SimpleTps is designed to be lightweight and easy to configure. Customize messages, request settings, and other options to fit your server.
